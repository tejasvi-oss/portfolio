# Web Dev 101 Workshop

This repository contains materials for a web development workshop.

## Project Structure

```
.
└── workshop/
    ├── .gitattributes
    ├── index.html
    └── style.css
```

## Getting Started

To view the workshop materials, simply open the `index.html` file in your web browser.

## Contents

- Basic HTML structure
- Simple CSS styling

## Usage

1. Clone or download this repository
2. Navigate to the `workshop` directory
3. Open `index.html` in your preferred web browser
